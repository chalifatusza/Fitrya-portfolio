## Deployment & Database Setup

To view and manage certifications, make sure the `porto_certifications` table is initialized in Supabase.

### Step 1: Create Supabase Certifications Table
Go to your **Supabase Dashboard** -> **SQL Editor** -> **New Query**, and execute this schema script:

```sql
-- Create certifications table with porto_ prefix
CREATE TABLE IF NOT EXISTS porto_certifications (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  title TEXT NOT NULL,
  issuer TEXT NOT NULL,
  issued_at TEXT,
  credential_url TEXT,
  credential_id TEXT,
  image_url TEXT,
  image_name TEXT, -- Stores fileName for Supabase storage cleanup
  created_at TIMESTAMPTZ DEFAULT NOW()
);
```
